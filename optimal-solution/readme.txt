Instructions:

There are 5 main files viz.

main_multi_1 : main file for multilinear extension with budget = 1
main_multi_2 : main file for multilinear extension with budget = 2
main)sol_2 : main file for optimal solution with budget = 2
main_sol_3 : main file for 'non-nested' optimal solution with budget = 3
main_sol_3_nested: main file for nested optimal solution with budget = 3

There are 3 subfolders within the source folder viz:

im_functions = all influence maximization functions are in there
im_outputs = all outputs are in there
network_files = all network files used are in there
